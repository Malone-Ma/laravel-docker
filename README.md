# Laravel on Docker

非常感谢[laradock](https://github.com/LaraDock/laradock).基于laradock上做了一些精简

### 支持的容器

- **数据库**
    - MySQL
    - PostgreSQL
- **缓存**
    - Redis
- **WEB服务器**
    - Nginx
- **PHP**
    - PHP7-FPM
    - PHP7-CLI
    - Composer
    - Git
    - Node
    - Gulp
    - Webpack
    - Curl
    - Wget
